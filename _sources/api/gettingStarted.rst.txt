Getting Started with the Course
===============================

*Time: 15 min (Install 5G Toolkit) + 10 minutes (setup SDRs)*


The course utilizes the `5G Toolkit <https://gigayasawireless.github.io/toolkit5G/>`_ and Pluto-SDR for hardware
emulations in 5G networks. For individuals interested in hands-on programming assignments and eager to deepen their
understanding of wireless networks using various Software Defined Radios (SDRs), they can install the
`5G Toolkit <https://gigayasawireless.github.io/toolkit5G/>`_ and set up SDR drivers on their systems using the
following links:

--------

* Install 5G-Toolkit:
    * `Web Guide <https://gigayasawireless.github.io/toolkit5G/install.html>`__.
    * Videos Guide: `Windows <https://www.youtube.com/watch?v=BFOCwUYx92E>`__ OS | `Linux <https://www.youtube.com/watch?v=QuVvtV4LDoA>`__ OS.
